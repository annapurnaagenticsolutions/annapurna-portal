# Code of Conduct

## Our pledge

We are committed to providing a welcoming, inclusive, and professional environment for all contributors.

## Our standards

**Expected behavior:**
- Professional, respectful communication
- Constructive feedback and code reviews
- Focus on improving the project and documentation
- Welcoming to newcomers and people learning

**Unacceptable behavior:**
- Harassment, personal attacks, or discriminatory language
- Trolling, insulting, or derogatory comments
- Spam or self-promotion unrelated to the project
- Publishing others' private information without consent

## Enforcement

Violations can be reported to hello@annapurna.ai. Reports will be reviewed and addressed promptly.

## Scope

This code of conduct applies to all project spaces — GitHub issues, PRs, discussions, and any other communication channels.
