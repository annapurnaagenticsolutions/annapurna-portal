# Contributing

Thank you for your interest in contributing! This project is part of Annapurna Agentic Solutions.

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b my-feature`
3. Make your changes
4. Commit: `git commit -m "Add: description of change"`
5. Push: `git push origin my-feature`
6. Open a Pull Request

## Contribution types

- Bug fixes
- New features
- Documentation improvements
- Translations
- Test coverage improvements

## Before opening a pull request

1. Test your changes locally
2. Update documentation if needed
3. Keep commits focused — one feature per PR
4. Use clear commit messages

## Code style

- Follow existing code style in the project
- No personal data or secrets in code
- MIT licensed — your contributions are MIT licensed too

## Questions?

Email hello@annapurna.ai or open a GitHub Issue.
