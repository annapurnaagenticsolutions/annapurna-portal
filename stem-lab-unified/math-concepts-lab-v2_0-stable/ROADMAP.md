# Roadmap after v2.0

## Next recommended releases

### v2.1 — Class 1–5 interaction depth
- More object-based micro-interactions
- Better early word-problem scaffolding
- More counting, grouping, measuring, clock, and money practice

### v2.2 — Class 6–8 manipulatives
- Integer number-line controls
- Fraction and rational-number comparison tools
- Algebra balance visual interactions
- Geometry angle/area exploration

### v2.3 — Class 9–10 long-form exam examples
- More board-style worked examples
- More proof reason scaffolds
- More graph reading and construction drills
- More mistake analysis

## Current boundary

No printable worksheet pack is planned for v2.0. The product should remain a guided learning lab first.
