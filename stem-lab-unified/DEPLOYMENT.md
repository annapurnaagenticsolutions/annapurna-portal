# Deployment Instructions

1. Extract this ZIP.
2. Upload every file and folder to a GitHub repository root.
3. In GitHub, open **Settings → Pages**.
4. Select the deployment branch, usually `main`, and root folder `/`.
5. Save and open the published GitHub Pages URL.
6. The home page should load from `index.html`.
7. Test the three subject buttons: Math, Physics, and Chemistry.

Do not upload only `index.html`; the subject folders and stable launcher files are required.
